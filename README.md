## Features

View contact list with all contacts displayed in cards

Search contacts by name

Add new contacts through a popup

## Setup

Make sure you have Node.js installed (version 14 or higher).

npm install

npm run dev

Open http://localhost:5173 in your browser.


## Libraries Used

React - for building the UI components

Vite - for fast development and building

Font Awesome - for icons (via CDN)

## Design Choices

Used plain CSS

Components are separated with their own CSS files

Modal popup for adding contacts instead of inline form

Light blue background with white cards

Font Awesome icons with blue styling
